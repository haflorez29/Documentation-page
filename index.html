Introduccion

Javascript es un lenguaje de programacion, ya que este sigue una logica indicada por el programador, más conocido como el lenguaje de script para páginas web, este se introduce en un proyecto desde el html de la siguinte manera

Interno

<script> </script>

Externo

<script src= "main.jss"> </script>

Variables

Es un elemento que se emplea para almacenar y hacer referencia a otro valor, entiendela como una caja a la cual le das un nombre y un valor, se crean sin signos especiales ni espacios entre palabras y de declaran de la siguiente forma

Var nombre= "valor";

Tipos de variables

  • Sin valor
  • Numerica
  • Cadena de texto
  • Boolena
  • Elemento HTML
  • Arreglos
  • Objetos
  • Funciones

Operadores

Los operadores son uno de los elementos fundamentales en cualquier lenguaje de programación, ya que son los que nos permiten trabajar con variables y datos.

Existen 4 tipos de operadores

  • Operadorres aritmeticos
  • Operadores de asignacion
  • Operadores de cadena de texto
  • Operadores de comparacion
  • Operadores logicos

Arreglos

son el método más común y simple de almacenar datos. Técnicamente, un Array es un objeto intrínseco de JavaScript. Admite datos del tipo cadenas, números, objetos, arrays, true, false, null.

var familia= ["Alejandro","Maria","Carlos",5, true];

Funciones

Una función es un procedimiento en JavaScript—un conjunto de sentencias que realizan una tarea o calculan un valor. Para usar una función, debe definirla en algún lugar del ámbito desde el cual desea llamarla.

las funciones se usan mediante dos pasos, la declaracion y porteriormente la ejecucion de la misma funcion.

declaracion

function saludar (){

alert ("hola mundo");

}

ejecucion

saludar ()
